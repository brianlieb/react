Let's make sure the tools you need are set up on your system and you are all 
set for the course.

Here are the steps to try the example (using terminal or command prompt):

1. cd trialrun 
2. npm install
3. npm run build
4. npm start

At this point a browser should open to location http://localhost:3000. If it does not, point your browser to that URL. You should see the message

Welcome to React

if all went well.
